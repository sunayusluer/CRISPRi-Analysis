Xin library includes 80K guides with different purpose. one of the main chunks is 30K CRISPRi test library, where top 2 TSS site of 20 core essential genes with alternative TSS site in IPSC were tiled with guides targeting -200 to 300 bp around TSS. Similarly top 2 TSS of another 20 core essential genes using canonical TSS are also tiled for 500bp around TSS. Remaining library consists of guides tiling ~900 essential genes from 0 to 100bp after TSS.

LFC is calculated as described before: there are two biological repeats, which are two separate infections. Each infection was split into two screens after first passage and kept as technical replicates.The final analyses done with merging all reads together represented here as Av_LFC
